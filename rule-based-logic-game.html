What a rule-based system is

It's a chain of if-then instructions a person decided ahead of time. Here we'll use a simplified version of NEWS2 — the early-warning score used across the NHS. Every band and threshold below was set by humans, not learned from data.

So what did the engine actually do?

📋
It only followed instructions. Each line was an if-then rule a human wrote. No pattern was learned, no data was needed beyond this one patient.
🔁
It's deterministic. Run it 100 times with the same observations and you get the same answer 100 times — that's why you can trust and audit it.
🔍
It's fully transparent. You can point to the exact rule that fired. Compare that with machine learning, where the "rule" is buried inside the data it learned from.
